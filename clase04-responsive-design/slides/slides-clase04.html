Diseñamos primero para móviles:

📐 MEDIA QUERIES BÁSICAS

Las reglas que adaptan tu CSS:

/* Estilos base (móvil) */ .navegacion { flex-direction: column; } /* Tablet y más grande */ @media (min-width: 768px) { .navegacion { flex-direction: row; } } /* Desktop */ @media (min-width: 1024px) { .navegacion { gap: 2rem; } }

/* ✅ Viewport obligatorio */ <meta name="viewport" content="width=device-width, initial-scale=1.0"> /* ✅ Media query para móvil */ @media (max-width: 767px) { nav ul { flex-direction: column; gap: 0.5rem; } nav a { padding: 1rem; display: block; } }

✨ EFECTOS HOVER BÁSICOS

Dale vida a los elementos interactivos:

/* Botón con hover suave */ .btn { background-color: #3498db; transition: all 0.3s ease; } .btn:hover { background-color: #2980b9; transform: translateY(-2px); box-shadow: 0 5px 15px rgba(52, 152, 219, 0.3); }

🎯 transition hace el cambio suave

🎯 FOCUS EN FORMULARIOS

Mejor accesibilidad y experiencia:

input { border: 2px solid #ddd; transition: border-color 0.3s ease; } input:focus { outline: none; border-color: #3498db; box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.1); }

💡 Accesibilidad = navegación por teclado

🌊 TRANSICIONES SUAVES

La clave de interfaces modernas:

/* Aplicar a todos los elementos interactivos */ .btn, .card, nav a, input { transition: all 0.3s ease; } /* Card con efecto elevation */ .card:hover { transform: translateY(-5px); box-shadow: 0 10px 30px rgba(0, 0, 0, 0.1); }

🎯 0.3s es el timing perfecto

🧪 TESTING RESPONSIVE

Cómo asegurarte de que funciona:

DevTools:

  • F12 → Toggle device toolbar
  • Probar iPhone, iPad, Desktop
  • Redimensionar manualmente
  • Verificar sin scroll horizontal

Dispositivos reales:

  • Tu teléfono móvil
  • Tablet si tienes
  • Live Server + IP local
  • GitHub Pages

🎯 Si funciona en móvil, funciona en todos lados

⚠️ ERRORES COMUNES

❌ Evita:

  • Olvidar viewport meta tag
  • Texto menor a 16px
  • Botones muy pequeños
  • Scroll horizontal
  • Efectos exagerados

✅ Mejor:

  • Viewport en todas las páginas
  • Mínimo 16px para texto
  • Botones de 44px mínimo
  • max-width: 100% en imágenes
  • Efectos sutiles
