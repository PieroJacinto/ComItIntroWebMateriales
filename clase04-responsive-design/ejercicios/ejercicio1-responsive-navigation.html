🎯 Ejercicio 1: Navegación Responsive

Objetivo: Hacer que esta navegación funcione perfectamente en móvil

📋 Tareas a Completar:

  1. Agregar viewport meta tag en el <head>
  2. Media query para navegación móvil:
    • Cambiar flex-direction a column
    • Reducir gap entre elementos
    • Hacer links más grandes para tocar
  3. Media query para botones móviles:
    • Padding más grande
    • Consideraar display: block
  4. Media query para servicios:
    • Cambiar de horizontal a vertical
  5. Probar en DevTools vista móvil

✅ Cómo Verificar que Está Correcto:

  • En móvil: navegación vertical y fácil de usar
  • En desktop: navegación horizontal como antes
  • Botones grandes y fáciles de tocar en móvil
  • Servicios en columna en móvil
  • NO scroll horizontal en ningún tamaño
