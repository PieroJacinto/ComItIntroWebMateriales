🏗️ HTML = HYPERTEXT MARKUP LANGUAGE

📝 HTML es el ESQUELETO de la web

  • 🧱 HTML = Estructura
  • 🎨 CSS = Diseño
  • ⚡ JavaScript = Funcionalidad

📱 Como construir una casa:

  • HTML = Paredes, techo, puertas
  • CSS = Pintura, decoración
  • JS = Electricidad, plomería

🔬 ANATOMÍA DE UNA ETIQUETA HTML

<h1>Mi título</h1> ↑ ↑ ↑ Apertura| Cierre Contenido <img src="foto.jpg" alt="Mi foto"> ↑ ↑ Atributo Atributo

💡 Siempre abrir y cerrar etiquetas

🔗 Atributos dan información extra

🏠 ESTRUCTURA BÁSICA HTML

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <title>Mi Página</title> </head> <body> <!-- Tu contenido aquí --> </body> </html>

🧠 head = información para el navegador

👁️ body = lo que ve el usuario

🏷️ ETIQUETAS QUE USAMOS HOY

🏠 Estructura:

<header> <main> <footer>

📰 Contenido:

<h1> <h2> <p> <ul> <li>

🔗 Enlaces:

<a href="...">Enlace</a>

🖼️ Imágenes:

<img src="..." alt="...">

📋 Formularios:

<form> <input> <textarea>

🏠 ESTRUCTURA SEMÁNTICA

<header> <!-- Título del sitio, navegación --> </header> <main> <!-- Contenido principal --> </main> <footer> <!-- Pie de página --> </footer>

💡 Como una casa:

  • header = techo
  • main = habitaciones
  • footer = cimientos

🧭 NAVEGACIÓN TÍPICA

<nav> <ul> <li><a href="index.html">Inicio</a></li> <li><a href="sobre-mi.html">Sobre Mí</a></li> <li><a href="proyectos.html">Proyectos</a></li> <li><a href="contacto.html">Contacto</a></li> </ul> </nav>

🔗 Enlaces internos vs externos:

  • "index.html" = misma carpeta
  • "https://google.com" = otro sitio

📋 FORMULARIO BÁSICO

<form> <label for="nombre">Nombre:</label> <input type="text" id="nombre" required> <label for="email">Email:</label> <input type="email" id="email" required> <label for="mensaje">Mensaje:</label> <textarea id="mensaje" rows="5"></textarea> <button type="submit">Enviar</button> </form>

🔗 label conecta con input usando for="id"

🐙 ¿QUÉ ES GIT Y GITHUB?

Git = Control de Versiones

  • 💾 Guarda historial completo
  • 🔄 Vuelve a versiones anteriores
  • 🤝 Trabajo en equipo
  • 📊 Ve qué cambió cuándo

GitHub = La Nube

  • ☁️ Backup automático
  • 💼 Portfolio profesional
  • 🌐 Compartir código
  • 👀 Los empleadores lo ven

🎯 Como WhatsApp: Escribes → Envías → Queda guardado

⚡ COMANDOS GIT BÁSICOS

git status # Ver qué cambió git add . # Preparar archivos git commit -m "..." # Guardar con mensaje git push # Subir a GitHub git pull # Actualizar desde GitHub

🔄 Workflow diario:

  1. Hacer cambios en archivos
  2. git add .
  3. git commit -m "Descripción"
  4. git push
