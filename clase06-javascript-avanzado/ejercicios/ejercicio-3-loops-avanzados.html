✅ VERIFICACIÓN:

Si completaste todo correctamente, deberías haber usado:

• While: Para búsquedas y contadores dinámicos

• For...of: Para iterar valores sin necesidad del índice

• For tradicional: Cuando necesitas el índice específicamente

⚠️ CUIDADO CON WHILE:

Siempre asegúrate de que la condición del while eventualmente se vuelva false, o tendrás un bucle infinito que puede bloquear tu navegador.

Incluye siempre: Un contador que aumente, o una condición que cambie.

💡 Recordatorio de Sintaxis:

• While: while (condicion) { /* código */ }

• For...of: for (let item of array) { /* código */ }

• For tradicional: for (let i = 0; i < array.length; i++) { /* código */ }
