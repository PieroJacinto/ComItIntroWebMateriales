🎨 ¿QUÉ ES CSS?

CSS = Cascading Style Sheets

  • 🏗️ HTML = Estructura (el esqueleto)
  • 🎨 CSS = Presentación (la piel y ropa)
  • ⚡ JavaScript = Comportamiento (los músculos)

🏠 Como construir una casa:

  • HTML = Paredes y habitaciones
  • CSS = Pintura y decoración
  • JS = Electricidad y funcionalidad

🔧 FORMAS DE INCLUIR CSS

✅ CSS Externo (RECOMENDADO)

<link rel="stylesheet" href="css/styles.css">

⚠️ CSS Interno (solo para pruebas)

<style> body { color: red; } </style>

❌ CSS Inline (EVITAR)

<p style="color: red;">Texto</p>

¿Por qué externo?

  • Reutilizable
  • Organizado
  • Fácil de mantener

📝 SINTAXIS CSS

selector { propiedad: valor; otra-propiedad: otro-valor; } /* Ejemplo real */ h1 { color: #3498db; font-size: 2rem; text-align: center; }

💡 Partes importantes:

  • Selector: A qué elemento aplicar estilos
  • Propiedad: Qué aspecto cambiar
  • Valor: Cómo cambiarlo

🎯 SELECTORES BÁSICOS

/* Selector de elemento */ h1 { color: blue; } /* Selector de clase */ .mi-clase { background-color: red; } /* Selector de ID */ #mi-id { border: 2px solid green; } /* Selector descendiente */ nav ul { list-style: none; }

Especificidad: ID > Clase > Elemento

🌈 COLORES EN CSS

Formas de definir colores:

/* Hexadecimal (MÁS COMÚN) */ color: #3498db; color: #333333; /* Nombres básicos */ color: red; color: white; /* RGB */ color: rgb(52, 152, 219);

Colores profesionales:

#3498db #2c3e50 #e74c3c #27ae60

✍️ TIPOGRAFÍAS

body { font-family: Arial, sans-serif; font-size: 16px; line-height: 1.6; } h1 { font-size: 2.5rem; font-weight: bold; text-align: center; } .destacado { font-style: italic; text-decoration: underline; }

💡 Reglas importantes:

  • Mínimo 16px para texto legible
  • Máximo 2 fuentes por sitio
  • Line-height de 1.5 o 1.6

📦 MODELO DE CAJA

¡Todo elemento HTML es una caja!

.mi-caja { /* Contenido */ width: 300px; height: 200px; /* Padding (espacio interno) */ padding: 20px; /* Border (borde) */ border: 2px solid #333; /* Margin (espacio externo) */ margin: 20px; /* ¡IMPORTANTE! */ box-sizing: border-box; }

🎯 SIEMPRE usar box-sizing: border-box

Margin = Espacio externo | Padding = Espacio interno

🏗️ ESTRUCTURA CSS RECOMENDADA

/* ===== RESET Y BASE ===== */ * { margin: 0; padding: 0; box-sizing: border-box; } body { font-family: Arial, sans-serif; font-size: 16px; line-height: 1.6; color: #333; } /* ===== HEADER ===== */ /* ===== NAVEGACIÓN ===== */ /* ===== MAIN CONTENT ===== */ /* ===== FOOTER ===== */

🏠 HEADER Y NAVEGACIÓN

header { background-color: #2c3e50; color: white; padding: 1rem 0; text-align: center; } nav ul { list-style: none; display: flex; justify-content: center; gap: 2rem; } nav a { color: white; text-decoration: none; padding: 0.5rem 1rem; border-radius: 5px; }

📝 CONTENIDO PRINCIPAL

main { max-width: 800px; margin: 0 auto; padding: 2rem; } h2 { font-size: 2rem; color: #2c3e50; margin-bottom: 1rem; border-bottom: 2px solid #3498db; padding-bottom: 0.5rem; } p { margin-bottom: 1rem; text-align: justify; } img { max-width: 100%; height: auto; border-radius: 8px; }

📋 FORMULARIOS ATRACTIVOS

form { background-color: #f8f9fa; padding: 2rem; border-radius: 8px; margin: 2rem 0; } label { display: block; margin-bottom: 0.5rem; font-weight: bold; } input, textarea { width: 100%; padding: 0.75rem; border: 2px solid #bdc3c7; border-radius: 4px; margin-bottom: 1rem; }

🔘 BOTONES PROFESIONALES

button { background-color: #3498db; color: white; padding: 0.75rem 2rem; border: none; border-radius: 4px; font-size: 1rem; cursor: pointer; }

💡 Transitions hacen todo más suave

🦶 FOOTER PROFESIONAL

footer { background-color: #2c3e50; color: white; text-align: center; padding: 2rem; margin-top: 3rem; } footer p { margin-bottom: 0; }

💡 TIPS DE DISEÑO RÁPIDO

  • 🎨 Máximo 3-4 colores principales
  • 📏 Espaciado consistente (múltiplos de 8px)
  • ✨ Transiciones suaves (0.3s ease)
  • 📱 Texto legible (mínimo 16px)
  • 🎯 Jerarquía clara (h1 > h2 > p)
  • 🔘 Botones que parecen botones

🚫 Menos es más - no sobrecargar

🚫 ERRORES COMUNES A EVITAR

❌ No hacer:

  • Demasiados colores
  • Texto muy pequeño (<16px)
  • Sin espaciado (todo pegado)
  • CSS inline en HTML
  • No usar box-sizing

✅ Mejor así:

  • Máximo 3 colores
  • 16px+ para texto
  • Margin y padding apropiados
  • CSS en archivo separado
  • Siempre box-sizing: border-box
