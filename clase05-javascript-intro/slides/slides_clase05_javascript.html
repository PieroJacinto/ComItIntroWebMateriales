📚 DEMO: Arrays Básicos

🎬 Abrir: ejemplo2-arrays_basicos.html

Arrays = listas organizadas de datos

// ✅ Creando arrays let frutas = ["manzana", "banana", "naranja"]; // ✅ Accediendo a elementos console.log(frutas[0]); // "manzana" console.log(frutas[1]); // "banana" // ✅ Propiedades útiles console.log(frutas.length); // 3

🎯 ¡Los índices empiezan en 0!

📚 ARRAYS: Métodos Básicos

let tareas = ["estudiar", "ejercicio"]; // ➕ Agregar al final tareas.push("cocinar"); console.log(tareas); // ["estudiar", "ejercicio", "cocinar"] // ➖ Quitar del final let ultimaTarea = tareas.pop(); console.log(ultimaTarea); // "cocinar" // 🔍 Buscar elemento console.log(tareas.indexOf("estudiar")); // 0 console.log(tareas.includes("limpiar")); // false // 🔗 Unir en string console.log(tareas.join(" - ")); // "estudiar - ejercicio"

🎯 Piensa: ficha personal con datos organizados

🔄 DEMO: For Loops

🎬 Abrir: ejemplo4-for_loop_tradicional.html

For loops = repetir acciones de forma controlada

let frutas = ["manzana", "banana", "naranja"]; // ✅ For loop tradicional for (let i = 0; i < frutas.length; i++) { console.log("Fruta " + (i + 1) + ": " + frutas[i]); } // 📤 Resultado: // Fruta 1: manzana // Fruta 2: banana // Fruta 3: naranja

🎯 Estructura: inicio, condición, incremento

🔄 FOR LOOPS: Anatomía

for (let i = 0; i < 5; i++) { console.log("Vuelta número: " + i); } // ① ② ③ // │ │ │ // │ │ └── Incremento (i++) // │ └────────── Condición (i < 5) // └──────────────── Inicio (let i = 0)
⚠️ CUIDADO: i++ no i-- (loop infinito)

💡 La base de toda repetición en programación

🚨 ERRORES COMUNES

❌ Confundir [] y {}
let persona = ["Juan", 25]; // ❌ Array let persona = {nombre: "Juan", edad: 25}; // ✅ Objeto
❌ Índices (empiezan en 0)
let frutas = ["manzana", "banana"]; console.log(frutas[1]); // ❌ "manzana"? NO! console.log(frutas[0]); // ✅ "manzana"
❌ For loops infinitos
for (let i = 0; i < 10; i--) // ❌ Infinito! for (let i = 0; i < 10; i++) // ✅ Correcto
