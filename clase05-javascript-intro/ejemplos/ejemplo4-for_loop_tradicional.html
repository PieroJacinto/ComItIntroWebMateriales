🔄 For Loop Tradicional

1. 🤔 ¿Por Qué Necesitamos Loops?

💡 Problema: ¿Qué pasa si queremos mostrar cada elemento de un array? ¿Escribimos todo a mano?
// Forma manual (😫 MUY repetitiva) let frutas = ["manzana", "banana", "naranja", "kiwi"]; console.log(frutas[0]); // manzana console.log(frutas[1]); // banana console.log(frutas[2]); // naranja console.log(frutas[3]); // kiwi // ¿Y si tenemos 100 frutas? 😱
💡 Solución: ¡Los loops nos permiten repetir acciones automáticamente!

2. 🏗️ Estructura del For Loop

for (inicialización; condición; incremento) { // código a repetir }
1. Inicialización: let i = 0 (empezamos en 0)
2. Condición: i < array.length (mientras sea menor que el tamaño)
3. Incremento: i++ (sumamos 1 en cada vuelta)
// Ejemplo básico for (let i = 0; i < 3; i++) { console.log("Vuelta número: " + i); } // Resultado: // Vuelta número: 0 // Vuelta número: 1 // Vuelta número: 2

3. 📝 For Loop con Arrays

let colores = ["rojo", "verde", "azul"]; // Recorrer todo el array for (let i = 0; i < colores.length; i++) { console.log("Color " + i + ": " + colores[i]); } // Resultado: // Color 0: rojo // Color 1: verde // Color 2: azul
💡 i es el índice que va cambiando (0, 1, 2...) y colores[i] accede a cada elemento.

5. 💼 Ejemplos Prácticos

// Sumar todos los números de un array let numeros = [10, 20, 30]; let suma = 0; for (let i = 0; i < numeros.length; i++) { suma = suma + numeros[i]; } console.log("Suma total: " + suma); // 60 // Contar elementos que cumplen una condición let edades = [15, 25, 17, 30, 16]; let mayores = 0; for (let i = 0; i < edades.length; i++) { if (edades[i] >= 18) { mayores = mayores + 1; } } console.log("Mayores de edad: " + mayores); // 2
