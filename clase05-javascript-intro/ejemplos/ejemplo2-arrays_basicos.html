📝 Arrays Básicos

1. 📚 ¿Qué son los Arrays?

💡 Analogía: Un array es como una estantería con cajones numerados. Cada cajón puede guardar un elemento.
// Crear un array de frutas let frutas = ["manzana", "banana", "naranja"]; // Cada elemento tiene una posición (índice) que empieza en 0 // Posición: 0 1 2 // ["manzana", "banana", "naranja"]

🗂️ Visualización del Array:

[
0
"manzana"
1
"banana"

"naranja"
]

2. 🛠️ Crear Arrays

// Array vacío let listaVacia = []; // Array de nombres let nombres = ["Ana", "Carlos", "María"]; // Array de números let numeros = [10, 25, 30, 45]; // Array mixto (diferentes tipos) let mixto = ["Juan", 25, true, "programador"];
💡 Los arrays pueden contener strings, números, booleans, ¡incluso otros arrays!

3. 🎯 Acceder a Elementos

let frutas = ["manzana", "banana", "naranja"]; // Acceder por índice (posición) let primera = frutas[0]; // "manzana" let segunda = frutas[1]; // "banana" let tercera = frutas[2]; // "naranja" // Obtener la cantidad de elementos let cantidad = frutas.length; // 3 // Último elemento usando length let ultima = frutas[frutas.length - 1]; // "naranja"
💡 ¡Importante! Los índices empiezan en 0, no en 1. El primer elemento es [0].

4. 🔧 Métodos Básicos de Arrays

let colores = ["rojo", "verde"]; // Agregar al final con push() colores.push("azul"); // Resultado: ["rojo", "verde", "azul"] // Quitar del final con pop() let ultimo = colores.pop(); // ultimo = "azul" // Resultado: ["rojo", "verde"] // Buscar elemento con indexOf() let posicion = colores.indexOf("verde"); // 1 // Verificar si existe con includes() let existe = colores.includes("amarillo"); // false
💡 push() agrega, pop() quita, indexOf() busca, includes() verifica.
